# Drivers

Aggregation of all data driver modules. Each driver encapsulates a specific data store or messaging system for use across the media ecosystem.

## Sub-modules

- [**mysql-driver**](mysql-driver/README.md) — Spring Data JPA driver for MySQL 
- [**postgres-driver**](postgres-driver/README.md) — Spring Data JPA driver for PostgreSQL 
- [**redis-driver**](redis-driver/README.md) — Spring Data Redis driver 
- [**kafka-driver**](kafka-driver/README.md) — Spring Kafka driver
