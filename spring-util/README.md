# Spring Util

Aggregation of shared Spring Boot utility modules.

## Sub-modules

- [**security**](security/README.md) — Spring Security config, token auth, RBAC
- [**security-flux**](security-flux/README.md) — WebFlux variant of the security module
- [**spring-web**](spring-web/README.md) — Spring MVC web conventions with OpenAPI/Springdoc
- [**spring-pod**](spring-pod/README.md) — Kubernetes pod base config (health checks, metrics)
- [**webflux**](webflux/README.md) — WebFlux utility methods
